## Monblank Client for Java

### Configuring maven

Add maven repository to your pom:

        <repository>
            <id>monblank-java</id>
            <url>https://raw.github.com/appmetr/monblank-java/mvn-repo/</url>
        </repository>


Add dependency:

        <dependency>
            <groupId>com.appmetr</groupId>
            <artifactId>monblank-java</artifactId>
            <version>0.1</version>
        </dependency>
