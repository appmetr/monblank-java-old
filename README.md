## Monblank Client for Java

### Configuring maven

Add maven repository to your pom:

        <repository>
            <id>appmetr-repo</id>
            <url>http://dl.bintray.com/appmetr/maven</url>
        </repository>


Add dependency:

        <dependency>
            <groupId>com.appmetr</groupId>
            <artifactId>monblank-java</artifactId>
            <version>0.3.1</version>
        </dependency>
